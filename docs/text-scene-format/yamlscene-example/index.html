YAML 场景文件的一个示例

下面给出了一个简单但完整的场景的示例。该场景只包含一个摄像机和一个立方体对象。请注意，文件 必须 使用下面两行作为开头：

%YAML 1.1
%TAG !u! tag:unity3d.com,2011:

…才能被 Unity 接受。否则，由于导入过程会容忍遗漏，因此会尽可能为缺失的属性数据提供默认值。
